About FinWise PH

Smart, Clear, and Honest Financial Comparisons for Every Filipino

Our Mission

FinWise PH was built with one goal: to simplify financial decision-making for Filipinos by providing clear, accurate, and up-to-date information on transfer fees, processing times, and supported banks.

Filipinos rely heavily on digital wallets, bank transfers, and online services — yet fee information is often confusing, scattered, or outdated. FinWise PH presents everything in a clean, organized, and easy-to-understand way so users can make confident financial choices.

How FinWise PH Works

Dynamic Data Loading

All service details — fees, transfer times, and limits — are stored in structured JSON files and loaded through JavaScript for fast, efficient updates.

Smart Filtering

Users can quickly filter by category, wallet, bank, or service type to find exactly what they need without digging through long pages.

Local Storage Favorites

Your favorite financial services are saved in your browser, allowing easy access every time you visit the site.

Responsive Design

FinWise PH works seamlessly across all devices, ensuring clean readability, whether you're browsing on mobile, tablet, or desktop.

Accessibility First

Built using semantic HTML, proper ARIA labels, and keyboard navigation to ensure every user can access the platform comfortably.

Performance-Driven

Optimized assets, lazy-loaded images, and efficient scripts ensure fast loading speeds even on slow connections.

Technology Behind FinWise PH

HTML5

Accessible structure, clean semantics, and properly defined metadata for SEO.

CSS3

Responsive layouts using Flexbox and Grid, combined with a clean, modern dark theme.

JavaScript (ES6+)

Dynamic UI updates, JSON fetching, array filtering, and localStorage functionality.

JSON Data

All service information is kept in structured data files for easy updates and future scaling.

Note: FinWise PH is built using pure HTML, CSS, and JavaScript — no frameworks — demonstrating mastery of core web development principles.

About the Creator

FinWise PH was created by Charles Nephi Salva as part of a WDD project to help Filipino users navigate the growing number of digital financial services. The goal is to provide a reliable, easy-to-use platform that simplifies comparing fees, limits, and processing times.

Why financial services? Millions of Filipinos rely on digital wallets and bank transfers daily. This sector is full of hidden fees, confusing information, and inconsistent updates — making it the perfect subject for a clarity-focused web project.

Skills Demonstrated:

  • JSON-based data structuring
